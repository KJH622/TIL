# 기본 문법 배우기_2

## 숫자

숫자는 따옴표로 감싸지 않고 그대로 작성하면 된다.

즉, 따옴표로 감싼 숫자는 문자열이다.

### 지수표기법 (exponential notation)

매우 큰 숫자나 소수점 아래 자릿수가 많을 때 숫자를 표현하는 방식이다.

e 뒤에 나오는 숫자만큼 10의 거듭제곱을 하면 된다.

```
5e4; // 50000
5e-4; // 0.0005
```

2진법은 숫자 앞에 `0b`를 붙인다.

```
0b111; // 7
```

8진법은 숫자 앞에 `0` 또는 `0o`를 붙인다.

```
0o15; 13
```

16진법은 숫자 앞에 `0x`를 붙인다. 단, 숫자는 0부터 9까지이기 때문에 10 ~ 15는 a, b, c, d, e, f로 표현한다.

```
0x1c; // 28
```

### Nan

Nan은 Not a Number이다.

자료형은 number이다.

### typeof

값이 어떠한 자료형을 가지고 있는지 확인할 수 있다.

```
typeof 5; // "number"
typeof "5"; // "string"
typeof Nan; // "number"
```

## 문자열을 숫자로 바꾸기

하나의 자료형을 다른 자료형으로 바꾸는 함수들이 존재한다.

### parseInt

문자열을 숫자로 바꿀 수 있다.

ParseInt는 최대한 숫자로 변경

Number는 숫자로 변경이 불가능하면 `Nan` 출력

```
parseInt('5'); // 5
Number('5'); // 5

ParseInt('3.14'); // 3
ParseFloat('3.14'); // 3.14
Number('3.14'); // 3.14

ParseInt('3월'); // 3
Number('3월'); // Nan
```

### prompt()

직접 값을 입력할 수 있다.

```
typeof parseInt(prompt());
```

![prompt](img/grammer_1.png)

`parseInt()`에서 2진법 사용

```
parseInt(111, 2); // 7
```

`parseInt()`에서 10진법 사용

```
parseInt(111, 10); // 111
```

## 무한 값 알아보기

수학에서는 0으로 나누기가 성립하지 않는다.

```
2/0; // Infinity
typeof Infinity // "number"

Infinity - Infinity; // Nan
0/0; // Nan
```

Infinity는 무한의 의미를 나타내고 있다.

## 문자와 숫자 더하기

문자열과 다른 자료형을 더하면 다른 자료형이 문자열로 바뀐 후 문자열과 더해진다.

이를 형 변환이라고 한다.

```
'문자열' + 0; // "문자열0"
'1' + 0; // "10"
```

## 문자와 숫자 빼기

문자열과 숫자를 빼면 문자열이 숫자로 바뀐 후 계산된다.

```
'문자열' - 0; // Nan
```

## 부동소수점

```
0.3 - 0.1; // 0.19999999999998
```

이를 해결하는 간단한 방법은 실수를 정수로 바꿔서 계산한 뒤, 마지막에 다시 실수로 바꾸는 것이다.

```
(0.3 * 10 - 0.1 * 10) / 10; // 0.2
```
