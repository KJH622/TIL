# HTML이란?

### HTML

- **H**yper **T**ext **M**arkup **L**anguage의 약자로서 웹 페이지를 만드는 언어

- **Hyper Text**: 월등한 최고의 기능을 담고 있는 문서

- 대표적인 기능: **하이퍼링크**

    - 텍스트를 클릭하면 링크된 다른 웹 문서로 이동하는 문서 연결

- **Mark up**: 웹 문서에서는 특정 텍스트에 Mark하기 위하여 HTML 태그를 사용

```
<h1>제목</h1>
<p>내용이 들어올 자리이다.</p>
<a href="http://www.naver.com">네이버로 이동</a>
```

- `<h1>` 태그: 제목 표현

- `<p>` 태그: 글의 문단 표시

- `<a>` 태그: 다른 웹 문서로 이동

> **HTML은 태그로 이루어진 Markup 언어**

### CSS

- **C**ascading **S**tyle **s**heets의 약자로, HTML 문서를 꾸밀 때 사용하는 스타일 시트 언어

### JavaScript

- **Markup Language** - HTML, CSS

- **Programming Language** - JavaScript

### 태그 특징

- 태그는 **시작 태그** `<h1>` 와 **종료 태그** `</h1>` 로 이루어져 있다.

- 태그 사이에 콘텐츠를 선언한다.

    - 예) `<h1>제목</h1>` 에서 제목이 콘텐츠이다.

- 콘텐츠가 없는 태그는 종료 태그를 생략할 수 있다.

- 시작 태그 안에 있는 class는 HTML TAG의 속성이며 class 이외에도 다양한 속성이 있다.

- HTML 태그는 요소 또는 노드라고 부른다.

    - 태그 = 요소(Element) = 노드(Node)

## HTML 문서 기본구조

- 웹 페이지를 작성하려면 기복적으로 선언되어야 하는 것들이 있다.

- 이를 HTML 문서 기본구조라고 하며, 그 구조는 아래와 같다. 

    `! + Enter` 하면 기본 구조 출력 가능

    ```
    <!DOCTYPE html>
    <html lang="ko">
	    <head>
		    <meta charset="UTF-8">
		    <title>문서 제목</title>
	    </head>
	    <body>
		    <h1>HTML</h1>
		    <p>이번 시간에 배울 내용은 HTML이다.</p>
	    </body>
    </html>
    ```

**1. `<!DOCTYPE html>`**

- **도큐먼트 타입으로 현재 문서가 어떤 버전으로 작성되었는지 미리 표시**한다.

     `<!DOCTYPE>` 은 태그는 아니며 웹 문서 제일 처음에 표시한다.
    
**2. `<html lang="ko"> ~ </html>`**

- 웹 문서의 시작(`<html>`)과 끝(`</html>`)을 나타내는 태그이다.

- `lang="ko"` : **lang 속성은 웹 문서가 어떠한 언어로 작성되었는지 표시**하는 `<html> 태그의 속성` 이다.

    현재 `ko` 는 한국어로 작성되었다는 뜻이며, 이러한 속성은 웹 접근성 향상에 도움을 준다.

**3. `<head> ~ </head>`**

- `<html>` 태그의 자식(Child) 태그로, **HTML 문서 정보나 문서에서 사용할 외부 파일 링크를 넣는 구간**이며, 웹브라우저에 표시되지는 않는다.

    - **`<meta charset="UTF-8">`**

        - HTML 파일의 인코딩을 알려주는 태그로서, 쉽게 말하면 브라우저에게 이 문서를 어떻게 그리는지 알려주는 용도이다.

        - `<meta charset="UTF-8">` 태그를 입력해야지 한글, 특수문자들이 깨지지 않고 잘 출력된다.

**4. `<body> ~ </body>`**

- `<html>` 태그의 자식(Child)태그로, **HTML 문서 내용을 넣는 구간**이며, 실제 웹 브라우저 안에 표시된다.

- `<body>` 태그 안에 실제 문서 내용을 입력하여 웹 사이트를 만든다.

## HTML 주석

- 주석은 HTML 소스 내에 메모를 남기는 것을 의미한다.

    ```
    <!-- 메모내용 -->
    <h2>영양성분</h2>

    <!--
	    메모내용이 여러줄일 경우 
    -->
    <h2>영양성분</h2>
    ```

- 주석은 실제 웹 화면에 보이지 않는다.

- 웹 화면에 보이지 않으나, "소스보기"를 통하여 볼 수 있기 때문에 중요한 정보를 남겨서는 안된다.
