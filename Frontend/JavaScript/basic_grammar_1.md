# 기본 문법 배우기_1

## 세미콜론

자바스크립트는 하나의 명령이 끝날 때 세미콜론을 붙여도 되고 붙이지 않아도 된다.

그러나 붙이는 것을 권장한다.

```
console.log('Hello World!');
```

한 줄에 여러 명령을 넣고자 할 때는 아래와 같이 표현한다.

```
console.log('Hello, World!'); console.log('Hello, JavaScript!');
```

## 주석

### 한 줄 주석

주석이 한 줄인 경우 주석 앞에 `//`를 넣으면 된다.

### 여러 줄 주석

주석이 여러 줄인 경우 주석 앞에 `/* */`를 넣으면 된다.

## 들여쓰기

들여쓰기는 하나로 통일하는 것이 좋다.

## 자료형

값(value)은 프로그래밍이 조작할 수 있는 데이터를 의미한다.

이러한 값의 종류를 자료형(data type)이라고 한다.

### 문자열(string)

문자들이 하나 이상 나열되어 있는 것을 의미한다.

**문자열은 따옴표로 감싸서 사용한다.**

백틱(`)으로 표현할 경우 줄바꿈이 가능하다.

```
'Hello, World!';
"Hello, World!";
`Hello, World!`;
```

### typeof

`typeof`를 사용하면 해당 문자열의 자료형을 알려준다.

```
typeof '문자열'; //"string"
typeof ''; // "stirng"
typeof ' '; // "string"
```

### 문자열 연산자

문자열이 긴 경우에는 문자열을 쪼갠 뒤 다시 합친다.

이 때, +연산자를 사용한다.

```
'안녕하세요? ' + '강지현입니다.'
// "안녕하세요? 강지현입니다."
```
