# 유클리드 기하학

- 우리가 실생활에서 자주 사용하는 동그란 원이 유클리드 기하학 원의 넓이이다.

- 정점에서 일정한 거리에 있는 점의 집합을 의미한다.

### 공식

**r * r * 원주율**

# 택시 기하학

- 택시 기하학에서의 원은 두 대각선의 길이가 같은 마름모 모양의 정사각형이다.

- 원의 정의가 정점에서 일정한 거리에 있는 점의 집합을 의미하는데 택시 평면 위에서는 정점에서의 거리가 같은 점의 집합이 마름모 모양의 정사각형이 된다.

- 아래에 모눈종이 같은 택시 평면 위에 택시 기하학의 원이 그려진 그림을 첨부한다.

![택시 기하학](/mathematical_expression/img/taxicab_img.png)

### 공식

**2 * r * r**

## 풀이 방식

```
import math
r = int(input()) # 반지름

euclid = math.pi * r * r # 유클리드 기하학
taxicab = 2 * r * r # 택시 기하학

# f"{변수:형식}"
print(f"{euclid:.6f}")
print(f"{taxicab:.6f}")
```
