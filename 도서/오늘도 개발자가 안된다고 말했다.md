# 오늘도 개발자가 안된다고 말했다.

## 개발자의 일

### 프론트엔드 개발자

- **사용 언어** : HTML, CSS, 자바스크립트

- **프레임워크**

    - 부트스트랩 (HTML & CSS & 자바스크립트 묶어 놓은 프레임워크)

    - 제이쿼리 (자바스크립트를 쉽고 빠르게 적용할 수 있음)

    - 하지만 요즘은 페이지 단위가 아닌 UI 컴포넌트 단위로, 스크립트 개발하는 웹 애플리케이션이 많이 제작되고 있다.

    - 즉, React.js, Vue.js, Angular.js 등이 존재한다.

### 백엔드 개발자

- **사용 언어** : Java, PHP, Python, Node.js, C++, C#, Ruby 등

- 웹 서버는 MySQL, MongoDB, Oracle 등 DB 제품을 웹 서버 프레임워크와 연결하여 사용한다.

- **프레임워크**

    - **Java 기반의 Spring** (모듈화하기 좋은 성향. 클라우드 서비스에서 사용)

    - **PHP 기반의 Laravel**

    - **Node.js 기반의 Express.js**

    - **Python 기반의 Django** (연산이 빠른 프레임워크로 빅데이터 연산과 분석이 필요한 서비스에 많이 이용)

    - **Ruby 기반의 Ruby on Rails**

### 풀스택 개발자

- 프론트엔드 개발과 백엔드 개발이 가능한 개발자

- 프론트엔드와 백엔드 언어가 동일한 경우 여러가지의 프레임워크를 활용해서 효율적으로 서비스 구축이 가능하다.

- **MEAN Stack(민 스택)** : MongoDB, Express.js, Angular.js, Node.js

    - **MongoDB** : 데이터베이스에 사용

    - **Express.js** : 웹 서버 개발에 사용

    - **Angular.js** : 프론트엔드에 사용

    - **Node.js** : 서버 구축에 사용
