# 아스키 코드

- 아스키코드는 American Standard Code for Information Interchange의 약자로, 각 알파벳에 숫자를 대응시켜 놓은 것을 의미한다.

- 내장함수인 `ord()` 와 `chr()` 을 활용할 수 있다.

## ord() 함수

- 지정한 문자에 해당하는 아스키 코드 값을 반환한다.

```
print(ord('A')) # 65
print(ord('a')) # 97
```

## chr() 함수

- 지정한 아스키 코드 값에 해당하는 문자를 반환한다.

```
print(chr(65)) # 'A'
print(chr(97)) # 'a'
```
