# .find()

### 어떤 상황에서 쓰는가?

- **안전하게 찾기 위해**

    - 없는 문자/문자열도 자주 나올 때

    - 예외 처리를 원하지 않을 때

    - 즉, 찾는 문자가 없으면 조용히 실패하고 반환 값 -1이다.

# .index()

### 어떤 상황에서 쓰는가?

- **반드시 존재해야 하는 값 찾을 때**

    - 없는 값이면 오류를 나게 해서 debug하기 좋다.

    - 데이터가 반드시 존재한다고 알고 있을 때

    - 즉, 찾는 문자가 없으면 오류 발생하고 ValueError 예외가 반환된다.
